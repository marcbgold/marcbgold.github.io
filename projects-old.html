Virtual Pet Shelter

Description: Game where the player is in charge of running a shelter for virtual cats and dogs. Pets can be either organic or robotic with different stats for each, and will respond to player inputs. Don't let them get too sick, or the whole shelter will get shut down!

Tools Used: Java 8, Git, TDD, OOP

Video Game Review Website

Description: Website housing humorous video game reviews in the form of haikus. Uses Spring framework to dynamically generate individual review pages from a single HTML template.

Tools Used: Java 8, Spring, HTML5, Git, TDD, OOP

Hospital Management System

Description: App demonstrating a simple employee management system for a hospital. User can search by name or unique ID number, as well as add and remove employees from the roster. User can also display all employees with specific capabilities.

Tools Used: Java 8, Git, TDD, OOP
